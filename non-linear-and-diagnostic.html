Model Diagnostic

Functional form for continuous predictor

The only continuous variable is exper. So we made scatter plot for exper and ln_salavg. According to the scatter plot, we hypothesized there is non-linear relationship between exper and ln_salavg.

Polynomial regression

For model 1 (interaction gender*exper)

We added its power 2, 3, 4 and its centering power 2, 3, 4 seperately to our model 1 to create polynomial models. But the summary result indicates that there is no association between higher order of exper and ln_salavg. Alao, by using anova, we fail to reject our reg13. There is no need to use high order model to improve our reg13.

For model 2 (interaction gender*rank)

We use the same process to test whether polynomial regression could improve our model containing interaction between gender and rank. There is no need to use higher order of exper, neither.

Piecewise model

To make piecewise model, I choose exper = 7.5 and exper = 9 as knots according to the scatter plot.

For model 1, P-value of 2 splines is larger than 0.05 in result of piecewise model. Alao, by using anova, we fail to reject our model 1. So there is no need to use piecewise model to improve model 2.

For model 2, P-value of knot at exper = 9 is 0.001 which indicate coefficient of exper change significantly here. However, the adjusted R-square is far less than reg14. Therefore, our original model fits the data better.

Assumptions

For model 1 (interaction gender*exper)

Based on the diagnostic plots for model 1, we can see the constant variance assumptions hold well. The curves in residuals vs fitted plot are horizontal and bounce around 0 which indicates that constant variance is satisfied in our model. Scale-location plot shows the same thing. In addition, the same observation is close to the 0.5 cooks distance in residuals vs leverage plot, which means No.184 observation might be influential.

For model 2 (interaction gender*rank)

Diagnostic plots for model 2 show similar things, except from No.184 observation, other observations satisfy all of our assumptions very well. It might be influential point in the same time.
